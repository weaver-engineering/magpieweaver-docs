Terms used across this HLD, the ADR set, and companion documents, kept
consistent throughout.

| Term                               | Definition                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ---------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **MagpieEngine**                   | The component marshaling what context (character state, lore, scene notes, conditional rules) is available for the entities involved in a scene, given the current scene state. Owns the core entity/state data model (§12.7, `entity-state-schema.md`).                                                                                                                                                                                                                                                                                                        |
| **Weaver**                         | The component orchestrating the LLM and walking through a scene's SceneEvents to generate prose. Runs continuity linting and derives proposed state mutations from generation output.                                                                                                                                                                                                                                                                                                                                                                           |
| **BranchingDataStore**             | The abstract persistence interface (`read`/`write`/`list`/`commit`/`revert`) that decouples the UI and engine from physical storage mechanics (§6). Renamed from the earlier "FileStore" — the interface is not a generic file store; it's the branching, Git-backed data-store contract that `GitDataStore` implements.                                                                                                                                                                                                                                        |
| **GitDataStore**                   | The implementation of the BranchingDataStore contract backed by native Git, used across **all** client surfaces (Enterprise Cloud, Mobile, and Local Desktop alike) — with checksum-validated writes, branch-scoped caching, and cache-eviction rules tied to specific Git operations (§6, §12.5, ADR-016). There is no separate desktop-specific implementation; `GitDataStore` runs native Git directly regardless of host. (An earlier draft posited a distinct `NativeGitFileStore` implementation for Local Desktop — that's been dropped as unnecessary.) |
| **Scene Director (mode)**          | The interactive authoring mode in which an author directs AI-agent character "actors" through a scene, using controls (`[Cut]`, `[Action]`, `[From <Event>]`, `[From the top]`, `[Wrap]`) and direct prose editing (§7.2, §10.1).                                                                                                                                                                                                                                                                                                                               |
| **Data Entry (mode)**              | The plain-editor authoring mode for entity fields (characters, lore, etc.), persisted continuously via write-ahead deltas, committed to Git only on `[Publish]` (§7.1).                                                                                                                                                                                                                                                                                                                                                                                         |
| **SceneEvent**                     | The smallest sequential unit within an in-flight Scene; the granularity at which state-change deltas and durability checkpoints are recorded (§4, §10.1, `entity-state-schema.md` §4).                                                                                                                                                                                                                                                                                                                                                                          |
| **Actor**                          | An AI-agent-driven character participating in Scene Director mode, generating in-character performance during a take.                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **EntityStateAttribute**           | An author-defined, immutable-`dataType` definition of a piece of state that can change over the course of the narrative (e.g. `isOrdained: boolean`). No attributes exist by default (`entity-state-schema.md` §2).                                                                                                                                                                                                                                                                                                                                             |
| **EntityStateAttributeValue**      | The current, live in-memory value of a given `EntityStateAttribute` for a given entity (`entity-state-schema.md` §3).                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **EntityStateAttributeValueDelta** | A record of one attribute's change (old → new) for one entity, scoped per SceneEvent and aggregated per Scene; `old: null` represents initial state (`entity-state-schema.md` §4).                                                                                                                                                                                                                                                                                                                                                                              |
| **Chronology**                     | The model of narrative time as a branching/mergeable sequence of chronological events, independent of final story order (§9).                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **PlotPoint**                      | A hierarchical unit of plot information (may contain other PlotPoints); may reference many `EntityStateAttribute`s. Full schema owned by the MagpieEngine HLD.                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Arc**                            | A grouping of all scenes involving a given entity (a character, an object, etc.), used as a query axis (§8).                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **Condition object**               | The (not yet defined) mechanism for gating context inclusion — e.g. `WorldLoreItem.scope`, conditional character facts — on an `EntityStateAttribute`'s current value. Owned by the MagpieEngine HLD.                                                                                                                                                                                                                                                                                                                                                           |
| **ADR**                            | Architectural Decision Record — a single approved decision with rationale and consequences, tracked in the companion ADR document(s).                                                                                                                                                                                                                                                                                                                                                                                                                           |
