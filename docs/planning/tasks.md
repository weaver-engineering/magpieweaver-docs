# Task Register

The single source of traceability for all Magpie Weaver work. Every branch and
PR maps to a task recorded here. See [Ways of Working](../ways-of-working.md)
for the process and reference conventions.

## Status legend

- **Proposed** — defined, not yet started.
- **Spec** — spec PR open or in review.
- **Tests** — failing-tests review open or in review (implementation repo).
- **Impl** — implementation review open or in review (implementation repo).
- **Done** — all gates merged.
