# Task Register

The single at-a-glance index of all Magpie Weaver work. Each task has its own
file (see [Ways of Working → Planning documents](../ways-of-working.md#planning-documents));
this register links to them and tracks status. Every branch and PR maps to a
task recorded here.

## Status legend

- **Proposed** — defined, not yet started.
- **Spec** — spec PR open or in review.
- **Tests** — failing-tests review open or in review (implementation repo).
- **Impl** — implementation review open or in review (implementation repo).
- **Done** — all gates merged.
